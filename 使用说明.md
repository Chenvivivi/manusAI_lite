# ManusAI 使用说明

## 🚀 快速启动

### 第一步：启动后端服务

打开终端，进入 backend 目录：

```bash
cd backend
python main.py
```

或者直接双击 `backend/start.bat` 文件（Windows）

你会看到：
```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
```

**保持这个终端窗口打开！**

### 第二步：打开前端页面

在浏览器中打开 `frontend/index.html` 文件

## 💬 如何使用

1. **发送消息**
   - 在底部输入框输入文字
   - 点击发送按钮（或按 Enter 键）
   - 你的消息会显示在右边（蓝色）
   - 后端返回的消息会显示在左边（深色）

2. **查看后端日志**
   - 每次发送消息，后端控制台会打印：
   ```
   ==================================================
   [2026-03-08 14:45:30] 收到用户消息:
   任务ID: 1
   消息内容: 你好
   --------------------------------------------------
   调用 DeepSeek deepseek-reasoner 模型...
   思维链内容: 用户在打招呼，我应该友好地回应...
   模型回复: 你好！我是 ManusAI，有什么可以帮助你的吗？
   ==================================================
   ```

3. **切换任务**
   - 点击左侧任务列表中的任务
   - 每个任务有独立的聊天记录

4. **创建新任务**
   - 点击左侧顶部的"+ 新建任务"按钮

5. **查看文件**
   - 点击消息中的文件标签
   - 右侧会滑出文件预览面板
   - 点击遮罩层或关闭按钮关闭

## 🔧 技术实现

### 后端 (FastAPI + DeepSeek)

- **接口地址**: `POST http://localhost:8000/api/chat`
- **功能**: 
  - 接收前端消息
  - 调用 DeepSeek Reasoner 模型生成回复
  - 在控制台打印模型返回结果（方便调试）
  - 返回模型回复和思维链内容
  - 维护多轮对话上下文
- **CORS**: 已配置允许跨域请求

### 前端 (Vue.js 3)

- **发送消息**: 使用 `fetch` API 调用后端接口
- **显示响应**: 将后端返回的内容显示为 AI 消息（左侧）
- **错误处理**: 如果后端未启动，会显示友好的错误提示

## 📝 API 文档

后端启动后，可以访问自动生成的 API 文档：

- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

## ⚠️ 注意事项

1. **必须配置 DeepSeek API Key**，否则无法调用模型
2. **必须先启动后端**，否则前端发送消息会失败
3. 后端默认运行在 `http://localhost:8000`
4. 如果端口被占用，可以修改 `backend/main.py` 中的端口号
5. 前端使用 Vue 3 CDN，无需安装 npm 依赖
6. DeepSeek API 按 token 计费，请注意使用量

## 🐛 调试技巧

- 打开浏览器开发者工具（F12）查看网络请求
- 查看后端控制台的打印信息
- 检查 Console 中是否有错误信息
