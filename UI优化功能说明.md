## 1. 📄 自动生成来源文件

### 功能说明
- ✅ 每次生成回答时，自动创建一个 Markdown 格式的来源汇总文件
- ✅ 文件包含所有搜索引擎的结果和链接
- ✅ 点击文件标签可在右侧面板查看
- ✅ 文件内容支持 Markdown 渲染
- ✅ 链接可点击跳转

### 文件内容示例

```markdown
# 搜索来源汇总

**用户问题**: 分析 Vue 和 React 的区别

**生成时间**: 2026-03-08 16:30:00

---

## 搜索 Vue.js 的特性和优势

**搜索引擎**: Tavily, Serper, Qianfan

### 各引擎直接答案

**[Tavily]**: Vue.js 是一个渐进式 JavaScript 框架...

**[Serper]**: Vue.js 3.0 带来了 Composition API...

**[Qianfan]**: Vue.js 是一款用于构建用户界面的渐进式框架...

### 搜索结果

1. **Vue.js 官方文档** `[Tavily]`
   - 链接: [https://vuejs.org/](https://vuejs.org/)
   - 摘要: Vue.js 是一个用于构建用户界面的渐进式框架...

2. **Vue 3 新特性详解** `[Serper]`
   - 链接: [https://example.com/vue3](https://example.com/vue3)
   - 摘要: Vue 3 引入了 Composition API...
```

### 使用方法
1. 发送消息并等待回答生成
2. 在 AI 回复下方会出现文件标签（如 `来源_20260308_163000.md`）
3. 点击文件标签
4. 右侧面板滑入，显示文件内容
5. 可以点击文件中的链接访问原始来源

## 2. 📋 复制消息内容

### 功能说明
- ✅ 每条消息旁边都有复制按钮
- ✅ 点击即可复制消息内容到剪贴板
- ✅ 复制成功后显示提示
- ✅ 支持复制用户消息和 AI 回复

### 使用方法
1. 将鼠标悬停在任意消息上
2. 点击消息右下角的 📋 复制按钮
3. 看到 "✓ 已复制" 提示
4. 可以粘贴到任何地方使用

### 样式特点
- 按钮默认半透明，悬停时高亮
- 复制成功后右上角显示绿色提示
- 提示 2 秒后自动消失

---

## 3. ⬛ 停止生成功能

### 功能说明
- ✅ 发送消息后，发送按钮变为停止按钮
- ✅ 点击停止按钮可中断回答生成
- ✅ 停止后显示 "回答已暂停" 提示
- ✅ 停止后可以继续发送新消息

### 使用方法
1. 发送消息后，输入框右侧的按钮变为红色 "⬛ 停止"
2. 如果想中断生成，点击停止按钮
3. 当前消息会显示 "⚠️ **回答已暂停**"
4. 可以继续发送新消息

### 技术实现
- 使用 `AbortController` 中断 fetch 请求
- 捕获 `AbortError` 显示暂停提示
- 自动重置生成状态

### 按钮状态
- **发送按钮**（蓝色）：未生成时显示
- **停止按钮**（红色）：生成中显示
- **禁用状态**：输入框为空时发送按钮禁用

## 4. ◀▶ 侧边栏收起/展开

### 功能说明
- ✅ 左侧任务列表可以收起
- ✅ 收起后只显示切换按钮
- ✅ 展开后显示完整任务列表
- ✅ 平滑动画过渡

### 使用方法
1. 点击左侧边栏右上角的 ◀ 按钮
2. 侧边栏收起，宽度变为 50px
3. 聊天区域自动扩展
4. 点击 ▶ 按钮可重新展开

### 样式特点
- 收起/展开带有平滑动画（0.3秒）
- 切换按钮悬停时放大
- 按钮位置固定，易于点击
