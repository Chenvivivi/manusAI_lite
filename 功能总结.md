# 🎉 ManusAI 功能总结

## 项目概述

ManusAI 是一个基于 DeepSeek Reasoner 的智能助手系统，集成了 Planner-Executor 架构和真实搜索引擎，能够自动规划任务、调用工具、搜索信息并生成高质量回答。

### 2. 后端架构

#### API 接口
- ✅ `/api/chat/stream`：流式聊天接口
- ✅ `/api/tools`：工具列表接口
- ✅ `/api/test/stream`：测试流式输出

#### 三层智能体架构

##### Planner（规划者）
- **模型**：DeepSeek Chat
- **功能**：将用户查询拆解为子任务
- **输出**：子任务列表（JSON 格式）
- **任务类型**：
  - `tool_call`：工具调用任务
  - `synthesis`：综合分析任务

##### Executor（执行者）
- **功能**：执行 Planner 规划的子任务
- **能力**：
  - 分析任务类型
  - 调用工具
  - 管理依赖关系
  - 收集执行结果

##### Reasoner（推理者）
- **模型**：DeepSeek Reasoner
- **功能**：基于工具执行结果生成最终回答
- **特点**：
  - 流式输出
  - 包含思维链
  - Markdown 格式

## 📈 工作流程示例

### 用户输入
```
分析 Vue 和 React 的区别
```

### Planner 规划
```json
{
  "subtasks": [
    {
      "id": 1,
      "type": "tool_call",
      "tool_name": "web_search",
      "description": "搜索 Vue.js 的特性和优势",
      "parameters": {
        "query": "Vue.js 特性 优势 2026",
        "max_results": 5
      }
    },
    {
      "id": 2,
      "type": "tool_call",
      "tool_name": "web_search",
      "description": "搜索 React 的特性和优势",
      "parameters": {
        "query": "React 特性 优势 2026",
        "max_results": 5
      }
    },
    {
      "id": 3,
      "type": "synthesis",
      "description": "对比分析 Vue 和 React 的区别",
      "dependencies": [1, 2]
    }
  ]
}
```

### Executor 执行
1. 执行子任务 1：搜索 Vue.js
   - 调用 Tavily 搜索
   - 返回 5 条结果
   
2. 执行子任务 2：搜索 React
   - 调用 Tavily 搜索
   - 返回 5 条结果

3. 执行子任务 3：综合分析
   - 收集前两个任务的结果

### Reasoner 生成
基于搜索结果，生成详细的对比分析：

```markdown

- **Vue**: 渐进式框架，核心库只关注视图层...
- **React**: 专注于构建用户界面的库...

## 2. 模板 vs JSX

...
```
