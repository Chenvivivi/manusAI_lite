# 🔍 搜索工具集成说明

## ✅ 已完成的功能

### 1. 多搜索引擎支持

实现了三种主流搜索引擎的集成：

#### 🎯 Tavily（推荐）
- **特点**：专为 AI 优化的搜索引擎
- **优势**：返回结果已针对 LLM 优化，包含直接答案
- **API 文档**：https://docs.tavily.com/
- **获取 API Key**：https://tavily.com/

#### 🔍 Serper
- **特点**：Google 搜索 API
- **优势**：快速、便宜，2500 次免费额度
- **API 文档**：https://serper.dev/
- **获取 API Key**：https://serper.dev/

#### 🇨🇳 百度千帆
- **特点**：百度搜索 API
- **优势**：中文搜索效果好，每日 100 次免费额度
- **API 文档**：https://cloud.baidu.com/doc/qianfan-api/
- **获取 API Key**：https://qianfan.cloud.baidu.com/

### 2. 智能引擎选择

系统会按优先级自动选择可用的搜索引擎：
1. **Tavily**（最优先，AI 优化）
2. **Serper**（次优先，Google 搜索）
3. **百度千帆**（备选，中文搜索）
4. **Mock 模拟**（开发测试用）

如果所有引擎都不可用，会返回模拟数据用于测试。

### 3. 工具执行结果传递

完整的数据流：

```
用户输入
  ↓
Planner（规划子任务）
  ↓
Executor（执行工具调用）
  ↓
搜索引擎（返回搜索结果）
  ↓
整合结果（格式化为上下文）
  ↓
DeepSeek Reasoner（基于搜索结果生成回答）
  ↓
流式返回给前端
```

## 🚀 使用步骤

### 1. 配置 API Key

编辑 `backend/.env` 文件，至少配置一个搜索引擎的 API Key：

```bash
# DeepSeek API（必需）
DEEPSEEK_API_KEY=your_deepseek_api_key

# 搜索引擎 API（至少配置一个）
TAVILY_API_KEY=your_tavily_api_key      # 推荐
# SERPER_API_KEY=your_serper_api_key    # 可选
# QIANFAN_API_KEY=your_qianfan_api_key  # 可选
```

### 2. 安装依赖

```bash
cd backend
python -m pip install -r requirements.txt --user
```

### 3. 启动服务

```bash
python main.py
```

### 4. 测试

打开前端页面，发送测试消息：

```
分析 Vue 和 React 的区别
```
