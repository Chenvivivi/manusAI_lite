# ⚡ 快速配置搜索引擎

## 🎯 推荐配置：Tavily（最简单）

### 为什么选择 Tavily？
- ✅ 专为 AI 优化，返回结果最适合 LLM
- ✅ 注册简单，无需信用卡
- ✅ 免费额度充足（1000 次/月）
- ✅ 响应速度快（1-2 秒）
- ✅ 包含直接答案（answer 字段）

### 5 分钟配置步骤

#### 1. 注册 Tavily 账号
访问：https://tavily.com/

点击 "Sign Up" 注册账号（支持 Google/GitHub 登录）

#### 2. 获取 API Key
登录后，在 Dashboard 页面会看到你的 API Key

格式类似：`tvly-xxxxxxxxxxxxxxxxxxxxxxxx`

#### 3. 配置到项目
编辑 `backend/.env` 文件：

```bash
# DeepSeek API（已有）
DEEPSEEK_API_KEY=你的deepseek_key

# 添加 Tavily API Key
TAVILY_API_KEY=tvly-xxxxxxxxxxxxxxxxxxxxxxxx
```

#### 4. 重启后端
```bash
# 如果后端正在运行，先停止（Ctrl+C）
# 然后重新启动
cd backend
python main.py
```

#### 5. 测试
打开前端，发送测试消息：
```
搜索 Python 3.12 新特性
```

查看后端日志，应该看到：
```
  → 尝试使用 Tavily 搜索...
  ✓ Tavily 搜索成功
```

## 🔄 备选方案

### 方案 1：Serper（Google 搜索）

**优点**：
- 基于 Google 搜索
- 免费额度更多（2500 次）
- 搜索结果质量高

**配置步骤**：
1. 访问 https://serper.dev/
2. 注册账号（支持 Google 登录）
3. 在 "API Keys" 页面获取 Key
4. 添加到 `.env`：
   ```bash
   SERPER_API_KEY=你的serper_key
   ```

### 方案 2：百度千帆（中文搜索）

**优点**：
- 中文搜索效果好
- 百度生态集成
- 每日免费额度

**配置步骤**：
1. 访问 https://qianfan.cloud.baidu.com/
2. 注册百度账号
3. 创建应用获取 API Key
4. 添加到 `.env`：
   ```bash
   QIANFAN_API_KEY=你的qianfan_key
   ```

如果暂时没有 API Key，系统会自动使用**模拟数据**：

```
  ⚠ 所有搜索引擎不可用，返回模拟数据
```

模拟数据可以用于：

- ✅ 开发调试

但模拟数据的回答质量较低，建议尽快配置真实的搜索引擎。

### 常见问题

#### 1. API Key 无效
**错误信息**：`Tavily API error: 401 - Unauthorized`

**解决方案**：
- 检查 API Key 是否正确复制
- 确认 API Key 前缀是 `tvly-`
- 重新生成 API Key

#### 2. 配置未生效
**错误信息**：`API key not found`

**解决方案**：
- 确认 `.env` 文件在 `backend/` 目录下
- 检查环境变量名称是否正确（`TAVILY_API_KEY`）
- 重启后端服务

#### 3. 网络超时
**错误信息**：`Tavily search timeout`

**解决方案**：
- 检查网络连接
- 尝试使用代理
- 切换到其他搜索引擎
