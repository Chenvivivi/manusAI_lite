# 🚀 ManusAI 快速开始指南

## 第一步：获取 DeepSeek API Key

1. 访问 https://platform.deepseek.com/
2. 注册/登录账号
3. 点击"API Keys"
4. 创建新的 API Key
5. 复制 API Key（格式：`sk-xxxxxxxxxxxxxxxx`）

## 第二步：配置 API Key

打开 `backend/.env` 文件，将 `your_api_key_here` 替换为你的真实 API Key：

```env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx
```

**保存文件！**

## 第三步：启动后端

### Windows 用户：
双击 `backend/start.bat` 文件

### 或者使用命令行：
```bash
cd backend
python main.py
```

看到这个就说明启动成功：
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

## 第四步：打开前端

在浏览器中打开 `frontend/index.html` 文件

## 第五步：开始聊天

1. 在底部输入框输入消息
2. 点击发送按钮
3. 等待 DeepSeek 模型回复（通常 2-5 秒）
4. 查看后端控制台的调试信息

## 常见问题

### Q: 提示"未配置 DEEPSEEK_API_KEY"
**A:** 检查 `backend/.env` 文件，确保 API Key 正确填写

### Q: 前端显示"连接后端服务失败"
**A:** 确保后端服务已启动，访问 http://localhost:8000 检查

### Q: 后端报错"API Key 无效"
**A:** 检查 API Key 是否正确，账户余额是否充足

### Q: 模型回复很慢
**A:** DeepSeek Reasoner 需要思考时间，通常 2-10 秒，复杂问题可能更长
