```python
class UnifiedSearchTool:
    async def search(self, query: str, max_results: int = 5, **kwargs):
        for engine_name, engine, api_key in self.engines:
            if not api_key:
                continue
            try:
                result = await engine.search(query, max_results, **kwargs)
                return result
            except Exception as e:
                continue
        return await self._mock_search(query, max_results)
```

## 🐛 已知问题

### 1. 搜索延迟
**问题**：搜索请求需要 1-3 秒
**影响**：整体响应时间增加
**计划**：后续版本添加缓存机制

### 2. API 额度限制
**问题**：免费额度有限
**影响**：超出额度后无法搜索
**解决**：配置多个搜索引擎作为备份

### 3. 网络依赖
**问题**：需要稳定的网络连接
**影响**：网络不稳定时可能失败
**解决**：自动重试和故障切换
