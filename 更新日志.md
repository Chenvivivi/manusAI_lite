# 更新日志

## 2026-03-08 - 版本 2.0

### ✨ 新增功能

#### 1. 思考过程展示
- ✅ AI 消息中显示 DeepSeek 模型的思维链（reasoning_content）
- ✅ 可折叠/展开的思考过程区域
- ✅ 独特的紫色主题样式，区分思考和回答
- ✅ 点击"思考过程"标题可切换显示/隐藏

#### 2. Markdown 渲染
- ✅ 支持完整的 Markdown 语法
- ✅ 标题自动分层（H1-H4），字体大小递减
- ✅ **粗体**、*斜体* 样式支持
- ✅ 代码块语法高亮（使用 highlight.js）
- ✅ 行内代码样式
- ✅ 列表（有序/无序）自动缩进
- ✅ 引用块样式
- ✅ 表格渲染
- ✅ 链接样式
- ✅ 分隔线

### 🔧 技术改进

#### 前端
- 引入 `marked.js` 进行 Markdown 解析
- 引入 `highlight.js` 进行代码高亮
- 添加 `renderMarkdown()` 方法
- 添加 `toggleReasoning()` 方法控制展开/折叠
- 消息对象新增 `reasoning` 和 `showReasoning` 字段

#### 后端
- 修复 Unicode 编码错误（emoji 显示问题）
- 添加 try-except 捕获打印错误
- 返回 `reasoning` 字段给前端

### 🎨 样式优化

- 思考过程区域：紫色主题，可折叠设计
- Markdown 标题：层级分明，字体大小递减
- 代码块：深色背景，语法高亮
- 列表：合适的缩进和间距
- 引用块：左侧紫色边框
- 表格：清晰的边框和表头样式

### 📝 使用说明

1. **查看思考过程**：
   - AI 回复中会显示"🤔 思考过程"区域
   - 点击标题可展开/折叠
   - 查看模型的推理过程

2. **Markdown 格式**：
   - 模型回复会自动渲染 Markdown 格式
   - 标题会自动分段和换行
   - 代码块会有语法高亮
   - 列表会自动缩进

### 🐛 Bug 修复

- 修复 Windows 控制台 emoji 显示导致的崩溃问题
- 修复消息显示不换行的问题
