## 🐛 如果还有问题

### 问题 1: 后端控制台仍然乱码
**解决方案**:
```powershell
# 在 PowerShell 中设置编码
chcp 65001
```

然后重启后端服务。

### 问题 2: 前端不显示子任务
**检查**:
- 打开浏览器开发者工具（F12）
- 查看 Console 是否有错误
- 查看 Network 标签，检查 API 返回的数据

### 问题 3: 流式输出不流畅
**可能原因**:
- 网络延迟
- DeepSeek API 响应慢
- 浏览器缓存问题（尝试硬刷新 Ctrl+F5）

## 📊 性能优化建议

1. **减少 max_tokens**: 如果回答太长，可以减少到 4000
2. **缓存对话历史**: 避免重复发送相同问题
3. **限制子任务数量**: Planner 最多规划 5 个子任务
